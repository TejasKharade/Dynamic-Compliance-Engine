DYNAMIC COMPATIBILITY & CONFIGURATION COMPLIANCE ENGINE
Test Rules Document
Purpose: Compact ruleset for feature-by-feature testing of extraction, Neo4j storage, compliance evaluation, impact analysis, and agent reasoning.

================================================================
1. PLATFORM OVERVIEW
================================================================

Supported hardware family:
- Dell Precision 3590
- Dell Precision 3591
- Dell Precision 7780

Components covered:
- BIOS
- Operating System
- Intel Chipset Driver
- Intel Graphics Driver
- NVIDIA Graphics Driver
- Dell Command Update
- Thunderbolt Docking Station

================================================================
2. BIOS RULES
================================================================

2.1 BIOS 1.5.0
- BIOS 1.5.0 requires Intel Chipset Driver version 7.0 or later.
- BIOS 1.5.0 supports Windows 11 23H2.
- BIOS 1.5.0 is not supported on Windows 10 22H2.

2.2 BIOS 1.6.2
- BIOS 1.6.2 requires Intel Chipset Driver >= 7.2.
- BIOS 1.6.2 requires Dell Command Update version 4.10 or newer.
- BIOS 1.6.2 is incompatible with NVIDIA Graphics Driver 31.0.15.4601.
- BIOS 1.6.2 is recommended only for devices that already run Windows 11 23H2.

2.3 BIOS 2.0.0
- BIOS 2.0.0 requires Intel Chipset Driver 7.4 or later.
- BIOS 2.0.0 requires Intel Graphics Driver 31.0.101.4502 or later.
- BIOS 2.0.0 requires Dell Command Update 4.11 or later.
- BIOS 2.0.0 supports Windows 11 24H2.
- BIOS 2.0.0 should not be installed if Thunderbolt Docking Station firmware is below 1.3.0.
- BIOS 2.0.0 is compatible with Dell Command Update 4.11 and 4.12.

================================================================
3. OPERATING SYSTEM RULES
================================================================

3.1 Windows 11 23H2
- Windows 11 23H2 requires Dell Command Update 4.10 or later.
- Windows 11 23H2 supports BIOS versions 1.5.0 and 1.6.2.

3.2 Windows 11 24H2
- Windows 11 24H2 requires Dell Command Update 4.11 or higher.
- Windows 11 24H2 requires Intel Chipset Driver 7.4 or later.
- Windows 11 24H2 requires BIOS 2.0.0 or later.
- Windows 11 24H2 is recommended for new deployments only.

3.3 Windows 10 22H2
- Windows 10 22H2 is supported only with BIOS 1.5.0.
- Windows 10 22H2 requires Intel Graphics Driver 30.0.101.3001 or later.
- Windows 10 22H2 should not be paired with Dell Command Update 4.12.

================================================================
4. DRIVER AND AGENT RULES
================================================================

4.1 Intel Chipset Driver
- Intel Chipset Driver 7.0 is compatible with BIOS 1.5.0 only.
- Intel Chipset Driver 7.2 is compatible with BIOS 1.6.2.
- Intel Chipset Driver 7.4 or later is required for BIOS 2.0.0 and Windows 11 24H2.

4.2 Intel Graphics Driver
- Intel Graphics Driver 31.0.101.4400 is incompatible with BIOS 1.6.2.
- Intel Graphics Driver 31.0.101.4502 or later is required for BIOS 2.0.0.
- Intel Graphics Driver 30.0.101.3001 or later is required for Windows 10 22H2.

4.3 NVIDIA Graphics Driver
- NVIDIA Graphics Driver 31.0.15.4601 is not compatible with BIOS 1.6.2.
- NVIDIA Graphics Driver 31.0.15.4700 or later is recommended for Precision 7780 systems.

4.4 Dell Command Update
- Dell Command Update 4.10 is required for Windows 11 23H2.
- Dell Command Update 4.11 is required for BIOS 2.0.0 and Windows 11 24H2.
- Dell Command Update 4.12 is compatible with BIOS 2.0.0.
- Dell Command Update 4.12 is not recommended on Windows 10 22H2.

================================================================
5. DOCKING STATION RULES
================================================================

- Thunderbolt Docking Station firmware 1.2.0 is not compatible with BIOS 2.0.0.
- Thunderbolt Docking Station firmware 1.3.0 or later is required for BIOS 2.0.0.

================================================================
6. CONDITIONAL RULES
================================================================

- If the device is running Windows 11 24H2, then BIOS must be 2.0.0 or later.
- If BIOS is 2.0.0, then Intel Chipset Driver must be 7.4 or later and Dell Command Update must be 4.11 or later.
- If Dell Command Update is 4.12, then the operating system must not be Windows 10 22H2.
- If the device is Precision 3590 and Windows 11 24H2, then Intel Graphics Driver 31.0.101.4502 or later is required.
